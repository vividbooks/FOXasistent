FOXasistent

Interní nástroj pro catering — směny, poloha a náklady na jídlo. GitHub Pages umí jen statické soubory; celá aplikace (Next.js) musí běžet na Vercelu nebo jiném Node hostu. Tuto stránku lze nastavit tak, aby okamžitě přesměrovala na nasazenou aplikaci (viz úprava v kódu stránky, proměnná APP_URL v <head>).

Chci sem „tu pravou“ stránku

  1. Nasaď fox-catering na Vercel (root directory fox-catering, stejné env jako lokálně).
  2. Zkopíruj produkční URL (např. https://…vercel.app).
  3. V repozitáři uprav docs/index.html — v <head> nastav var APP_URL = "https://…"; a pushni na main.

Po tomhle bude vividbooks.github.io/FOXasistent/ jen vstupní bod, který tě hodí do živé aplikace.
